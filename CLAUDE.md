# CM New — project rules

## One section, one tab
Always keep all variants/options/alternatives of the SAME page section in a SINGLE
HTML file (one tab). When the user asks for more alternatives of a section that
already has a file, ADD them as new `<DCArtboard>`s on that file's existing design
canvas — never create a second file for the same section. Group related directions
with `<DCSection>` inside the one file.

Current section files:
- `Grandmasters - All Options.html` — all Grandmaster section directions
- `Method - All Options.html` — all Method ("Diagnose. Study & Practice. Fix.") directions

## Design system
Visuals follow the bound Design System (ChessMood). Fonts: Plus Jakarta Sans
(display), Roboto (body), Schibsted Grotesk (numerals). Primary accent: #F57C00.
Use real ChessMood photography from `ds/`.
